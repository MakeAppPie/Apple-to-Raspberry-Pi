Popper's Penguins
By Steve Lipton  http//www.makeAppPie.com
Current Revision 2014 May 27

Description: Popper's penguins  is a data collection application using GUI and persistent
             Data in Python. It is the example application for these concepts for the 
             "From Apple to Raspberry Pi" series on the makeAppPie blog. It collects 
             a type of penguin and its current activity. 

Objective of the lessons:
			These lessons have the following objectives:
			*Understand the use of TKinter  tk and ttk widgets
			*Implement MVC in a Python application
			*Implement list processing
			*Implement persistent data with SQLite
			*implement with mySQL
			*create reports
			*create a histogram of data using tkinter
For more information about the lessons check the blog entries at the above URL.
